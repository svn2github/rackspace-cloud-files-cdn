Stores all of your WordPress media on Rackspace's Cloud Files CDN.

== Description ==

Allows any media in your WordPress uploads folder to be uploaded to Rackspace's Cloud Files CDN.

= Beta Version 0.0.1 =

I've tested this quite thoroughly. It looks to be very stable.

== Installation ==

1. Upload plugin to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
